Selectors and Filters

Selectors and filters provide a way of finding and extracting information from web pages.

Selectors

Selectors are used to select part of the web page using CSS syntax

For example, $('p') will select all of the p tags in a doc and return them as an array that can be further operated on.


Filters

Filters are used to further refine the results returned from selectors.

For example, $('p:first') will select the first paragraph in the returned set



$('document').ready(function() {
 // Callback function on ready
 // You can target by id, class or element
 $('code').css('background-color','aquamarine');

 // You can use advanced CSS selectors and mix in filtering
 $('ul > li:first').css('color','orange');

 // Or filter out a class
 $('div:not(.class)').css('color','orange');
});



Creating and Changing Page Content

$('document').ready(function() {
 // Create some new content
 // Notice that html() closes the <p> tag for you in output

 var newParagraph = $('<p>');
 newParagraph.append('Hey this is a string, added via jQuery.');
 $('#example').html(newParagraph);

 // Insert some content in the same tag using prepend() or append()
 $('#create').append(' - I just appended this text!');

 // Change existing content using html()
 $('#what').html('Why is jQuery Awesome?');
});



Event Handling with jQuery

jQuery normalizes an event structure across browers and provides cross-browser consistency for properties such as the event name, the page coordinates of the mouse, the element where the event originated along with many other elements related to the event, and information such as whether a meta-key or specific mouse button was pressed.

Mouse over me, go ahead.

// Event Handling Code
// We use the on() method with some custom callback functions
// The events are triggered only on the targeted element
$('#event-diagnostic').on('mousemove', onMouseOver);
$('#event-diagnostic').on('click', onMouseClick);
$('#event-diagnostic').on('mouseleave', onMouseLeave);

// We pass our callback functions the evt param, jQuery's cross browser event object
function onMouseOver(evt) {
$('#event-diagnostic').text('Mouse over fired.');
}
function onMouseClick(evt) {
$('#event-diagnostic').text('Mouse click fired.');
}
function onMouseLeave(evt) {
$('#event-diagnostic').text('Mouse leave fired.');
}

AJAX Made Simple.

One of jQuery's greatest strengths is how it simplifies working with AJAX. jQuery provides high-level wrapper functions for retreiving data, working with different data types, handling errors, and monitoring the AJAX request process.

You must either be using a development server for AJAX, or use Brackets Live Editor.

  • ajax() is used to perform generic AJAX requests
  • load() is used to load AJAX content directly into a page element
